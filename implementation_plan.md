# 잠코치 (SleepCoach) MVP 재개발 구현 계획

기존 코드를 바탕으로 **디자인 완성도(Premium Aesthetics)**와 **코드의 견고함(Robustness)**을 극대화하여, 상용 앱 수준의 하이엔드 MVP로 다시 개발합니다.

## 목표
- 주어진 `개발체크리스트_잠코치.md`의 STEP 1~4를 완벽하게 충족하는 완성도 높은 웹앱 구현.
- 사용자가 첫눈에 감탄할 수 있는 **프리미엄 다크 모드, 글래스모피즘, 마이크로 애니메이션** 적용.
- 유지보수와 확장이 용이한 모듈형 JavaScript 아키텍처 도입.

---

## Proposed Changes

### 1. UI/UX 및 디자인 시스템 개편 (style.css)
- **프리미엄 색상 팔레트**: 딥 네이비 배경에 부드러운 인디고/퍼플 네온 글로우를 적용하여 신비롭고 편안한 밤의 분위기를 연출합니다.
- **고급 글래스모피즘**: 단순히 투명도를 주는 것을 넘어, 빛 반사(Edge lighting)와 부드러운 그림자를 더해 입체감 있는 카드를 만듭니다.
- **마이크로 인터랙션**: 버튼 호버 시 부드럽게 떠오르는 효과, 입력 폼 포커스 시 네온 아웃라인, 화면 전환 시 매끄러운 Fade & Slide 애니메이션을 추가합니다.
- **타이포그래피**: 'Outfit'과 'Noto Sans KR'을 활용해 모던하고 가독성 높은 텍스트 계층 구조를 잡습니다.

### 2. 구조 및 마크업 개선 (index.html)
- **모바일 최적화 레이아웃**: 한 손으로 조작하기 편하도록 하단 내비게이션(Bottom Tab)을 고도화하고, 터치 영역(Tap Target)을 충분히 확보합니다.
- **입력 폼 컴포넌트화**: 시간 입력, 슬라이더, 토글 버튼을 시각적으로 더 직관적이고 조작하기 쉬운 형태로 재구성합니다.
- **대시보드 위젯화**: 분석 결과와 미션 카드를 위젯 형태로 배치해 한눈에 데이터가 들어오게 합니다.

### 3. 핵심 비즈니스 로직 리팩토링 (script.js)
- **모듈화**: 기존 통짜 스크립트를 State, Storage, Analysis, UI, AI API의 5개 모듈 객체로 명확히 분리하여 안정성을 높입니다.
- **분석 알고리즘 고도화**:
  - 수면 시간 부족 (8시간 미만)
  - 폰 사용량에 따른 컨디션 하락 (중앙값 기준)
  - 취침 시각 불규칙성 (표준편차 60분 이상)
  - 낮 졸림 빈도 (50% 이상)
  - 위의 로직을 엣지 케이스(예: 7일 미만 데이터일 때의 처리 방식)까지 고려하여 단단하게 구현합니다.
- **AI 코멘트 통합**: Gemini API를 호출하여 자연스러운 코멘트를 받아오되, API 실패나 키 미입력 시에도 매우 자연스러운 로컬 폴백(Fallback) 메시지를 제공하도록 에러 핸들링을 강화합니다.

## User Review Required

> [!IMPORTANT]  
> 이 계획은 기존 작성된 코드를 **완전히 덮어쓰고(Overwrite)** 처음부터 더 높은 품질로 재작성하는 것을 의미합니다. 기존 데이터 스키마(로컬 스토리지 구조)는 최대한 유지하겠지만, 디자인과 코드 구조는 크게 바뀔 것입니다.

## Verification Plan

### 수동 테스트 (Manual Verification)
1. **F1 (데일리 기록)**: 취침/기상 시각 입력 시 수면 시간이 정확히 계산되는지, 자정을 넘긴 시간도 잘 처리되는지 확인합니다.
2. **F2 (패턴 분석)**: 데모 데이터(7일치)를 주입한 뒤, 4가지 분석 규칙이 수학적으로 정확히 작동하여 카드로 노출되는지 확인합니다.
3. **F3 (오늘의 미션)**: 도출된 최우선 패턴에 맞는 미션이 정확히 매핑되어 표시되는지 확인합니다.
4. **UI/UX**: 데스크톱 크롬과 모바일 시뮬레이터에서 화면이 깨지지 않고 유려하게 전환되는지 확인합니다.
